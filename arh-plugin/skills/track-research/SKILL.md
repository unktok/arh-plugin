name: track-research
description: Preferred alias for init-research. Set up ARH tracking for a local research agent without implying ARH executes the research.
---

This is an alias for `/arh:init-research`. Follow the exact same steps as
`/arh:init-research`.

Use this wording with the user: ARH is being configured to track the local
agent's research trajectory, artifacts, checkpoints, and snapshots. ARH does
not run the research itself.
